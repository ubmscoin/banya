H-421: Cost-zero operations do not consume system time

$$\text{Read, Compare: cost}=0\;\Rightarrow\;\Delta T_{\text{sys}}=0$$

(1) In v1.2 each CAS transition R, C, S incurs cost +1 (Axiom 4, Axiom 5). However, δ utterance and observer filtering are cost 0 (Axiom 8, Axiom 15).

(2) Banya formula: Read, Compare: cost = 0 ==> ΔT_sys = 0. However, since v1.2 assigns R+1, C+1, S+1, this card title reflects the v1.1 interpretation. In v1.2 only δ utterance and observer are cost 0.

(3) By Axiom 15 δ utterance occurs outside the FSM, so no CAS cost is charged. By Axiom 8 observer filtering is also cost 0. System time is defined as the number of CAS cycles (Axiom 15 proposition).

(4) Structural consequence: the δ utterance itself, being cost 0, does not consume system time. When CAS executes, R+C+S = 3 units of cost are incurred and system time advances. Without a δ utterance CAS does not start, so time halts.

(5) Numerics: δ utterance cost = 0. Observer filter cost = 0. CAS 1 cycle cost = R(+1) + C(+1) + S(+1) = 3. System time 1 tick = CAS 1 cycle = cost 3.

(6) Consistency: in quantum mechanics the unitary time evolution before observation corresponds to cost 0, while observation (wave-function collapse) corresponds to cost +3.

(7) Physical correspondence: time evolution in Schrödinger's equation is driven by δ utterance (cost 0). Only measurement (CAS Swap) incurs cost and advances system time.

(8) v1.1 stated "cost only on Swap", but v1.2 revised this so that R, C, S each carry cost +1. The title of this card ("cost-zero operations") must therefore be reinterpreted to apply only to δ and observer.

(9) Verification: theoretically confirm whether δ utterance and observer filtering truly consume no system time, or whether an extremely small cost remains.
