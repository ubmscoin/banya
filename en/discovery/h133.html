$s = \text{SP bit count}/2$. Dividing the number of SP (superposition) bits by 2 yields the spin quantum number. The discreteness of the bit makes the $1/2$ unit necessary.

In Axiom 15 (8-bit structure), the SP bit sits in nibble 1 of the d-ring. Axiom 9 (complete-descriptor induction) fixes the SP bit's role. Because a bit is either 0 or 1, the minimum partition unit is $1/2$.

A d-ring bit takes only 0 or 1. 1 SP bit = spin $1/2$, 2 SP bits = spin $1$, 0 SP bits = spin $0$. Bit discreteness forces spin quantization, so continuous spin values are structurally impossible.

Since this is a structural correspondence, there is no numeric error — it is exact. Only spin $0, 1/2, 1, 3/2, 2$ is allowed.
