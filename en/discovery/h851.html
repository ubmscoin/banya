H-851: MRI = lock bit 정렬의 외부장 ball명

$$\omega_0 = \gamma B_0 \;\;\text{(lock 비트 = 핵스핀 → 외부장 정렬 → 라모 공명)}$$

Grade: B

MRI(자기ball명영상)는 핵스핀의 lock bit를 외부 자기장으로 정렬시킨 뒤, ball명 주파수의 RF 펄스로 lock bit를 교란하고, 이완 과정에서 방출되는 신호를 measurement하여 영상을 구성하는 기술이다.

공리 10(lock bit = 스핀), 공리 5(CAS Compare = ball명 조건), 공리 7(cost = RF 에너지)

공리 10: lock bit = 핵스핀 Status. 외부 자기장 $B_0$가 lock bit를 정렬(에너지 준위 분리). 공리 5: CAS Compare가 RF 주파수 $\omega$와 라모 주파수 $\omega_0 = \gamma B_0$를 compare ==> ball명 시 Swap(에너지 흡수). 공리 7: RF cost 흡수 ==> 이완 시 방출.

T1 이완: lock bit가 열평형 복귀(스핀-격자). T2 이완: lock bit 간 위상 상실(스핀-스핀). 경사자계: ball간 인코딩 ==> 영상 재구성. 대조도: T1/T2 차이 = 조직 구별.

임상 MRI: $B_0 = 1.5$~$3\;\text{T}$. 양성자 $\gamma/2\pi = 42.577\;\text{MHz/T}$. 해상도: $\sim 1\;\text{mm}$. fMRI: BOLD 대조도.

라모 주파수 $\omega_0 = \gamma B_0$ 정밀 실측 확인.
