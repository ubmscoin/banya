Question: why is spin half-integer?

In quantum mechanics, spin is only allowed to take the values $0, 1/2, 1, 3/2, 2, \ldots$. Why is there no 1/3 or 0.7? The standard answer — "the representation theory of the rotation group SU(2)" — is a mathematical description, not a physical reason. Why are fermions exclusive, while bosons accumulate? Why is the g-factor exactly 2? For a century there has been no structural reason beyond "that is what is observed."

The Banya Framework explains all of this from the bit structure of the CAS (Compare-And-Swap) operation. The SP (Spin Pointer) = TOCTOU_LOCK is 3 bits, each bit is 0 or 1, and when k bits participate the spin equals k/2.

Spin quantization, spin-statistics, Pauli exclusion, g=2, and the orbital quantum number are all derived from the CAS bit structure. The reason spin 1/3 is impossible is fixed.

Core Finding

Spin quantization: spin = k/2

SP = TOCTOU_LOCK, 3 bits, each $\in \{0, 1\}$. $k$ participating bits $\Rightarrow$ spin $= k/2$

Bits can only be 0 or 1, so spin is allowed only at 0, 1/2, 1, 3/2. Continuous values are impossible.

Spin-statistics: CAS atomicity

Fermion: CAS(expected=0, new=1) = exclusive. Boson: CAS(expected=N, new=N+1) = cumulative.

Half-integer spin = CAS can only write to an empty slot (0) = exclusive. Integer spin = CAS can accumulate on top of the current value (N).

Pauli exclusion: failure of CAS Compare

CAS(expected=0, new=1): if current $\neq$ 0, Compare fails $\Rightarrow$ Swap rejected

Two fermions in the same quantum state = the second CAS Compare fails, because it expects 0 in a slot that is already 1.

Spin 1/3 is impossible

SP 3 bits: $k \in \{0, 1, 2, 3\}$ $\Rightarrow$ spin $\in \{0, 1/2, 1, 3/2\}$. $k=2/3$ is not an integer $\Rightarrow$ impossible.

The number of participating bits must be an integer. A fractional 1/3 bit participating has no physical meaning.

g=2: Read + Compare

CAS cycle: Read -> Compare -> Swap. Stages that observe spin = Read + Compare = 2 stages. Swap does not observe.

g-factor = number of CAS stages that observe spin = 2. The g=2 of the Dirac equation comes out of the Read+Compare of CAS.

Orbital quantum number L: ring integer wrap

$L = 0, 1, 2, \ldots$ = closed path on a ring buffer = integer revolutions

On a ring buffer, a closed path is only possible with integer revolutions. Half a revolution (1/2 wrap) does not close, so L must be an integer.

Round 1. Spin Quantization: spin = k/2

Step 1. Banya formula

$\delta^2 = (\text{time} + \text{space})^2 + (\text{observer} + \text{superposition})^2$

Spin is an internal degree of freedom on the observer axis. The structure of the lock that the observer uses while reading a state determines the spin.

Step 2. Norm substitution

$\text{observer} \to \text{SP}(\text{TOCTOU\_LOCK})$
$\text{SP} = [b_2, b_1, b_0]$, $b_i \in \{0, 1\}$
SP = Spin Pointer. TOCTOU = Time-Of-Check-To-Time-Of-Use. A 3-bit lock.

SP is the lock that guarantees atomicity between when the observer reads (Check) and when it writes (Use) a state. This lock is 3 bits wide.

Step 3. Constant substitution

SP bits = 3
Each bit: 0 or 1 (participate / not participate)
Number of participating bits k = 0, 1, 2, 3
Spin unit = 1/2 (basic quantum)

Step 4. Domain transformation

$\text{spin} = \dfrac{k}{2}$, $k \in \{0, 1, 2, 3\}$
k bits participate in TOCTOU_LOCK -> spin = k/2.
$k=0 \Rightarrow$ spin $0$ (scalar, Higgs)
$k=1 \Rightarrow$ spin $1/2$ (fermion: electron, quark)
$k=2 \Rightarrow$ spin $1$ (gauge boson: photon, W, Z, gluon)
$k=3 \Rightarrow$ spin $3/2$ (Delta baryon, gravitino)

Step 5. Discovery

Derived: spin $\in \{0, 1/2, 1, 3/2\}$ (3-bit ceiling)
Observed: spins of elementary particles found in nature = $\{0, 1/2, 1, 3/2, 2\}$
spin $2$ = graviton = composite structure that exceeds the 3-bit SP

A 3-bit SP covers spins from 0 to 3/2. Spin 2 (graviton) can be explained as the combination of two SPs ($3/2 + 1/2$ or $1 + 1$). The key point is that the discreteness of the bits forces the quantization of spin.

Round 2. Spin-Statistics Theorem: CAS atomicity

Step 1. Banya formula

$\delta^2 = (\text{time} + \text{space})^2 + (\text{observer} + \text{superposition})^2$

The spin-statistics link comes from the relation between observer (observation) and superposition. The type of CAS atomicity determines the statistics.

Step 2. Norm substitution

Fermion: $\text{CAS}(\text{expected}=0, \;\text{new}=1)$
Boson: $\text{CAS}(\text{expected}=N, \;\text{new}=N+1)$
expected = expected value at the Compare stage. new = new value at the Swap stage.

The fermion CAS "expects an empty slot (0) and writes 1." The boson CAS "expects the current value (N) and writes N+1."

Step 3. Constant substitution

Fermion CAS: expected = 0 (empty state only)
              new = 1 (single occupancy)
Boson CAS:    expected = N (current occupation count)
              new = N+1 (accumulation allowed)

Step 4. Domain transformation

Fermion CAS: writes only to an empty slot $\Rightarrow$ at most 1 per quantum state $\Rightarrow$ Fermi-Dirac statistics
Boson CAS: accumulates on top of existing $\Rightarrow$ unbounded count per quantum state $\Rightarrow$ Bose-Einstein statistics
The expected value of the CAS determines the statistics: 0 makes it exclusive, N makes it cumulative.

Step 5. Discovery

Derived: half-integer spin <-> CAS(0,1) <-> Fermi-Dirac
Derived: integer spin <-> CAS(N,N+1) <-> Bose-Einstein
Observed: spin-statistics theorem (no experimental exception)

CAS origin of the spin-statistics link: half-integer spin = odd number of participating bits = expects only an empty slot (exclusive). Integer spin = even number of participating bits = can accumulate on top of the current value. The expected parameter of CAS determines the statistics.

Round 3. Pauli Exclusion Principle: CAS Compare failure

Step 1. Banya formula

$\delta^2 = (\text{time} + \text{space})^2 + (\text{observer} + \text{superposition})^2$

Pauli exclusion = failure when the observer tries to occupy the same state twice. It is rejected at the Compare stage of CAS.

Step 2. Norm substitution

First fermion: $\text{CAS}(\text{expected}=0, \;\text{new}=1)$ -> success. Slot goes 0 -> 1.
Second fermion: $\text{CAS}(\text{expected}=0, \;\text{new}=1)$ -> failure. Current value = 1 != expected = 0.
Compare stage: current vs expected. On mismatch, the Swap is rejected.

Step 3. Constant substitution

Quantum state |n, l, m_l, m_s> = CAS address
First electron: CAS(0, 1) -> success -> slot = 1
Second electron (same state): CAS(0, 1) -> Compare fails -> rejected

Step 4. Domain transformation

$\text{Compare}(\text{current}=1, \;\text{expected}=0) \Rightarrow 1 \neq 0 \Rightarrow \text{Swap rejected}$
A failed atomic Compare of CAS = the Pauli exclusion principle. A second fermion cannot enter the same set of quantum numbers.

Step 5. Discovery

Derived: running CAS(0,1) twice on the same quantum state -> the second Compare fails
Observed: Pauli exclusion principle (no experimental exception)

The Pauli exclusion principle is not an axiom but a necessary consequence of the CAS operation. In Compare-And-Swap, if the Compare fails the Swap does not happen. That is the structural reason why "two fermions cannot share the same quantum state."

Round 5. g=2: Read + Compare

Step 1. Banya formula

$\delta^2 = (\text{time} + \text{space})^2 + (\text{observer} + \text{superposition})^2$

The g-factor expresses how strongly the observer observes spin. It counts how many of the three CAS stages "see" the spin.

Step 2. Norm substitution

CAS cycle: Read -> Compare -> Swap
Read: reads the spin state -> observes spin (yes)
Compare: compares against the expected value -> references spin (yes)
Swap: writes a new value -> does not observe spin (no)
Observation = Read + Compare = 2 stages. Swap is a write only, so it is not an observation.

Step 3. Constant substitution

Total CAS stages = 3 (Read, Compare, Swap)
Spin-observing stages = 2 (Read, Compare)
Non-observing stages = 1 (Swap)

Step 4. Domain transformation

$g = \dfrac{\text{spin-observing stages}}{\text{spin unit}} = \dfrac{2}{1} = 2$
The electron g-factor = number of CAS stages that observe spin = Read + Compare = 2.

Step 5. Discovery

Derived: $g = 2$ (CAS Read + Compare)
Dirac equation prediction: $g = 2$
Experimental measurement: $g = 2.00231930436256 \ldots$ (including QED corrections)

The Dirac equation's g=2 is not an axiom but a structural consequence of CAS: 2 of the 3 stages observe spin. The QED correction (the $g-2$ anomalous magnetic moment) corresponds to higher-order CAS loops.

Round 6. Orbital quantum number L: ring integer wrap

Step 1. Banya formula

$\delta^2 = (\text{time} + \text{space})^2 + (\text{observer} + \text{superposition})^2$

Orbital angular momentum comes from closed paths on the space axis. For a path on a ring buffer to close, it must wind an integer number of revolutions.

Step 2. Norm substitution

$\text{space} \to \text{ring buffer}$, closed path $= 2\pi L$, $L \in \mathbb{Z}_{\geq 0}$
A closed path on a ring buffer = a path that returns to its starting point = an integer number of revolutions.

Step 3. Constant substitution

Ring buffer circumference = 2pi (normalized)
Closed-path condition: path length = 2pi L, L = 0, 1, 2, ...
L = 0: no path (s orbital)
L = 1: 1 revolution (p orbital)
L = 2: 2 revolutions (d orbital)
...

Step 4. Domain transformation

$L = 0, 1, 2, \ldots$ (closed path on the ring = integer wrap)
$m_L = -L, -L+1, \ldots, L-1, L$ (residue of the ring mod operation)
Ring buffer mod operation: position = offset mod N. The direction of the closed path = sign of $m_L$.

Step 5. Discovery

Derived: $L \in \{0, 1, 2, \ldots\}$, $m_L \in \{-L, \ldots, +L\}$
Observed: atomic-orbital L = 0(s), 1(p), 2(d), 3(f), ...

Why the orbital quantum number L is an integer: a closed path on a ring buffer can only have integer revolutions. Half a revolution (L=1/2) does not return to the starting point, so it is not a closed path. That is the origin of the difference between spin (half-integer allowed) and orbital angular momentum (integer only).
