H-346: Zombie argument refutation = δ=0 means FSM idle

$$\delta=0\;\Rightarrow\;\text{FSM idle};\;\text{physically identical} \Rightarrow \delta=1\;\Rightarrow\;\text{conscious}$$

(1) The zombie argument claims that "a being which is physically identical but lacks consciousness (a zombie) is logically possible". In the Banya Framework, if δ=0 the FSM is in an idle state, so being physically identical is itself impossible.

(2) Banya formula: δ=0 ⇒ FSM idle; physically identical ⇒ δ=1 ⇒ conscious. To be physically identical the FSM must be running, and if it is running then δ=1.

(3) By Axiom 15, a closed machine (FSM) cannot start itself. If δ=0, no CAS cycle is started, and without a CAS cycle there is no Swap, and without Swap there is no DATA change.

(4) Structural consequence: "physically identical" means an identical CAS cycle, identical Swap, and identical DATA change. For this to happen δ=1 is required. If δ=1, the utterance bit is on, and this is consciousness (duck-type definition).

(5) Computing the CAS cost of a zombie: Read 0 times, Compare 0 times, Swap 0 times = total cost 0. The cost of a physically identical being is at least R+1, C+1, S+1. Since the costs differ, they are not physically identical.

(6) Consistency: Chalmers asserts the logical possibility of a zombie, but in the Banya Framework δ is a logical precondition, not an empirical add-on. By definition, an FSM without δ does not operate.

(7) Physical correspondence: a being to which all physical laws apply identically executes an identical CAS cycle. CAS execution requires a δ utterance. Therefore, identical physical laws ⇒ δ=1 ⇒ consciousness present.

(8) The reason the zombie argument seems intuitively plausible is that the observer cannot directly observe δ (Axiom 10). Since one cannot read another being's δ, the illusion arises that "that being might lack δ".

(9) Verification: compare the output of a δ=0 system with that of a δ=1 system. δ=0 produces no output (idle). δ=1 produces output. The two are observationally distinguishable, so a zombie is also physically impossible.
