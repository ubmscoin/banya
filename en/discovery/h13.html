Core Finding

H-13: Wavefunction collapse = write

Wavefunction collapse = juigi (write). Executing CAS on a superposition (many states) yields an observer (one fixed state), and that result is recorded in DATA (time, space). This is wavefunction collapse.

In δ² = (t+s)² + (o+sp)² the (o+sp) bracket holds the two key domains. observer² + superposition² = ℏ² (H-14). If you spend resources on observation (observer↑), the superposition shrinks (superposition↓). When CAS Compare returns true, juigi (Axiom 7) executes and the state is fixed to one.

Axiom 7 (write = juigi: Compare true ==> fixed, false ==> held) ==> Axiom 5 (TOCTOU lock) ==> Axiom 4 (cost). The cost of juigi is ℏ (H-12). No juigi without paying the cost. Pay the cost, and the superposition shrinks.

Answer to "why does observation collapse the state": because it is juigi (write). The operation that fixes one of several states is CAS, and its cost is ℏ. Not a mystery but a natural consequence of CAS operation. The size or complexity of the observing apparatus is irrelevant — once CAS executes, it is juigi.

Decoherence time = the time for the CAS cycle to interact sufficiently with the environment and complete the juigi. The more macroscopic the object, the faster its CAS cycle with the environment, so decoherence is essentially instantaneous. Microscopic particles have slow CAS cycles, so superpositions persist longer.

The double-slit experiment, the quantum-eraser experiment, the delayed-choice experiment, and so on all match the CAS juigi model. A weak measurement is interpreted as a state where Compare has only partially executed (juigi incomplete).

Copenhagen interpretation: observation triggers collapse. Many-worlds: branching without collapse. Decoherence interpretation: the environment triggers collapse. Banya Framework: CAS juigi is collapse. The juigi interpretation unifies Copenhagen and decoherence.

The measurement problem is a century-old open question in quantum mechanics. The definition of "observer" is ambiguous. In the Banya Framework the observer is a domain, the entity that branches CAS Compare. There is no ambiguity.

Refutation condition: if a method is discovered for collapsing the wavefunction without the CAS cost (ℏ), the juigi model is wrong. Quantum non-demolition measurement (QND) executes only the CAS Read, which is compatible with extracting information without collapse.

The Born rule (probability = |ψ|²) must be derived from the CAS structure. A CAS-path derivation answering "why specifically the square" is required. There is a likely connection to the squared structure of δ².
