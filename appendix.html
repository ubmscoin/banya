적중오차 1% 이내 + 물리적 정당성 확보. 끝난 것도출 완료 + 측정값과 일치각 부속 보고서
발견새로운 공식/관계식이 확인됨. 재대입 가능한 인자오차 1% 이내가설 라이브러리 D 시리즈
가설구조적 대응은 확인됐으나 정량적 증명이 아직 미완구조 확인, 수식 미완가설 라이브러리 H 시리즈
진행착수했으나 완료되지 않음. 추가 작업 필요진행 중각 부속 보고서 미완 섹션
대기 (고유 예측)도출은 끝남. 실험 검증을 기다리는 중아무도 측정 못 한 값을 먼저 도출고유 예측 보고서
적중:  도출 완료 + 측정값 일치 -> 끝. 더 이상 작업 없음
발견:  새 공식 확인 -> 라이브러리에 등록 -> 다음 라운드에 재대입
가설:  구조는 보인다 -> 아직 수식이 미완 -> 다음 라운드에서 정밀화
진행:  착수했지만 안 끝남 -> 계속 작업
대기:  답을 아무도 모른다 -> 프레임이 먼저 말한다 -> 실험이 확인해줄 때까지 대기

왜 고유 예측(대기)이 가장 중요한가: 적중과 발견은 "이미 답을 알고 맞춘 것 아니냐"는 공격을 받을 수 있다. 가설과 진행은 아직 진행 중이다. 하지만 고유 예측은 아직 아무도 모르는 값을 먼저 말해놓은 것이다. 나중에 실험이 이 값을 확인하면 반박이 불가능하다. 힉스 보손이 1964년에 예측되고 2012년에 확인된 것처럼, 중력파가 1916년에 예측되고 2015년에 확인된 것처럼.
