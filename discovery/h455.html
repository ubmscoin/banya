H-455. 양자 비파괴 측정 = Read만 수행, Compare-Swap 미실행

핵심 발견

H-455: 양자 비파괴 측정 = Read만 수행, Compare-Swap 미실행

$$\text{QND}: \quad [H, A] = 0 \;\Longleftrightarrow\; \text{Read}(A)\text{만 실행},\; \text{Compare-Swap 우회}$$

등급: B

양자 비파괴(QND) 측정에서 관측량 $A$의 고유 상태가 해밀토니안 $H$와 교환한다($[H,A]=0$). 반야 프레임: Read가 해당 DATA 슬롯을 읽되, Compare 단계에서 "이미 고유 상태"이므로 Compare true ==> Swap해도 동일 상태로 덮어씀(멱등). 실질적으로 Read만 수행한 것과 동치. 상태 교란 없이 정보 획득.

공리 2(R==>C==>S 파이프라인에서 S가 멱등), 공리 7(Compare true이되 Swap이 항등)

공리 2(CAS 파이프라인) ==> 공리 7(Compare ==> Swap, 단 $|k\rangle → |k\rangle$ 멱등) ==> 공리 3(DATA 이산 ==> 고유 상태 안정) ==> 공리 13(ECS 인덱스 불변)

QND 측정은 CAS의 예외가 아니라 "Swap이 항등 연산이 되는 특수 경우"이다. $[H,A]=0$의 CAS 번역: Read 대상 슬롯과 시간 진화 연산자의 작용 슬롯이 직교 ==> Compare 후 Swap해도 DATA 불변. 반복 측정 시 동일 결과 보장(사영 측정 멱등성).

광자 수 QND(Nogues 1999): 공동 내 광자 수 반복 측정 시 동일 결과. CAS: Read(n) ==> Compare(n=n) ==> Swap(n==>n) 멱등.

QND 측정 실험(Braginsky, Nogues 1999) 결과와 정합.

양자 비파괴 측정(QND, Braginsky 1980), 공동 QED 광자 수 측정

반복 QND 측정에서 결과 불변 확인. 반박: $[H,A]=0$인데 반복 측정 결과가 변하는 경우.

QND 가능 관측량의 CAS 분류 체계. 근사 QND(back-action evasion)의 부분 Compare 모델.

재대입 용도: 재대입 용도: H-454(약한 측정) 극한 경우. H-442(불확정성) QND 예외. H-446(측정 문제) 멱등 측정.
