핵심 발견

H-640: 폴라리톤 = 광자-물질 혼성 CAS 모드

$$\omega_\pm = \frac{\omega_c + \omega_0}{2} \pm \sqrt{g^2 + \left(\frac{\omega_c - \omega_0}{2}\right)^2} \;\leftrightarrow\; \text{CAS 비용 혼성 모드}$$

등급: C

폴라리톤은 광자(CAS 비용파)와 물질 여기(엑시톤, 포논, 플라즈몬)의 혼성 모드이다. 강결합 영역에서 CAS 비용이 광자와 물질 사이를 라비 진동한다.

공리 4(비용파 = 광자), H-639(플라즈몬)/H-630(포논)

공리 4(CAS 비용파 + 물질 비용 = 혼성), 공리 2(CAS 결합 = 라비 진동), H-639(플라즈몬), H-630(포논)

엑시톤-폴라리톤 = FSM 쌍(엑시톤) + 광자의 혼성. 포논-폴라리톤 = 격자 진동 + 광자. 폴라리톤 BEC = 보손적 혼성 모드의 응축.

마이크로캐비티 라비 분리: $\sim 5\text{--}50\;\text{meV}$.

폴라리톤 실험(마이크로캐비티)과 정합.

폴라리톤, 라비 분리, 엑시톤-폴라리톤 BEC, 광자 마이크로캐비티

마이크로캐비티 폴라리톤 BEC 실험(2006) 검증.

폴라리톤 초유체의 CAS 메커니즘.
