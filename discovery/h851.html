H-851. MRI = lock 비트 정렬의 외부장 공명

핵심 발견

H-851: MRI = lock 비트 정렬의 외부장 공명

$$\omega_0 = \gamma B_0 \;\;\text{(lock 비트 = 핵스핀 → 외부장 정렬 → 라모 공명)}$$

등급: B

MRI(자기공명영상)는 핵스핀의 lock 비트를 외부 자기장으로 정렬시킨 뒤, 공명 주파수의 RF 펄스로 lock 비트를 교란하고, 이완 과정에서 방출되는 신호를 측정하여 영상을 구성하는 기술이다.

공리 10(lock 비트 = 스핀), 공리 5(CAS Compare = 공명 조건), 공리 7(비용 = RF 에너지)

공리 10: lock 비트 = 핵스핀 상태. 외부 자기장 $B_0$가 lock 비트를 정렬(에너지 준위 분리). 공리 5: CAS Compare가 RF 주파수 $\omega$와 라모 주파수 $\omega_0 = \gamma B_0$를 비교 ==> 공명 시 Swap(에너지 흡수). 공리 7: RF 비용 흡수 ==> 이완 시 방출.

T1 이완: lock 비트가 열평형 복귀(스핀-격자). T2 이완: lock 비트 간 위상 상실(스핀-스핀). 경사자계: 공간 인코딩 ==> 영상 재구성. 대조도: T1/T2 차이 = 조직 구별.

임상 MRI: $B_0 = 1.5$~$3\;\text{T}$. 양성자 $\gamma/2\pi = 42.577\;\text{MHz/T}$. 해상도: $\sim 1\;\text{mm}$. fMRI: BOLD 대조도.

라모 주파수 $\omega_0 = \gamma B_0$ 정밀 실측 확인.

핵자기공명, MRI, 라모 세차, T1/T2 이완, 스핀

MRI 50년 이상 임상 사용. 노벨의학상 2003.

lock 비트 모델에서 T1/T2 이완 시간의 정량 도출.
