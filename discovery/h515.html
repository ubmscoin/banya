H-515. 펜타쿼크 = 5-CAS 번들의 FSM 결합

핵심 발견

H-515: 펜타쿼크 = 5-CAS 번들의 FSM 결합

$$P_c = qqqq\bar{q},\quad \text{5-CAS 번들}:\; \text{FSM 닫힘 유지}$$

등급: C

펜타쿼크($qqqq\bar{q}$)는 5개 쿼크/반쿼크가 색 싱글렛을 형성하는 이국적 하드론이며, 5개 CAS 번들이 FSM 닫힘 조건을 만족하는 결합이다. LHCb(2015)에서 $P_c(4380)$, $P_c(4450)$ 발견. 이들은 FSM 조합 규칙(H-501)의 비표준 모드이다.

공리 12(FSM 닫힘 = 색 싱글렛), 공리 14(FSM 노름 = 질량), H-501(스펙트럼 조합)

공리 12(FSM 닫힘: $qqqq\bar{q}$도 색 싱글렛 가능 ==> 허용), 공리 14(5-쿼크 FSM 노름 합 ==> 질량 $\sim 4{-}5\;\text{GeV}$), 공리 4(CAS 비용 ==> 결합 에너지: 높은 CAS 비용 ==> 불안정), 공리 2(CAS 5중 결합 구조)

펜타쿼크는 분자형($\bar{D}^{(*)}\Sigma_c^{(*)}$) 또는 컴팩트 5-쿼크로 해석 가능. FSM 관점: 두 FSM 닫힌 클러스터의 약한 결합(분자) vs 하나의 5-FSM 닫힌 영역(컴팩트). 높은 CAS 비용 ==> 넓은 폭(불안정).

$P_c(4312)$, $P_c(4440)$, $P_c(4457)$ (LHCb 2019). 폭 $\Gamma \sim 10{-}200\;\text{MeV}$.

LHCb 발견 확인. 질량은 $\bar{D}^{(*)}\Sigma_c^{(*)}$ 문턱 근처 ==> 분자형 지지.

펜타쿼크(LHCb 2015/2019), 이국적 하드론, 하드론 분자, 매력 바리온

LHCb/Belle II에서 추가 펜타쿼크 탐색. 바닥 쿼크 펜타쿼크 예측.

분자 vs 컴팩트 구분의 FSM 판별. 비매력 펜타쿼크(경쿼크만) 가능성. 결합 메커니즘 정량화.

재대입 용도: 재대입 용도: H-501(스펙트럼) 이국적 확장. H-516(이국적 하드론) 분류. H-504(안정성) 불안정 모드.
