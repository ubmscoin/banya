H-942. 반정수 스핀 = 사원수 720도 회전 항등성

핵심 발견

H-942: 반정수 스핀 = 사원수 720도 회전 항등성

$$R(360°)\,|\psi_{\text{fermion}}\rangle = -|\psi_{\text{fermion}}\rangle, \quad R(720°)\,|\psi_{\text{fermion}}\rangle = +|\psi_{\text{fermion}}\rangle$$

등급: A (사원수 명제 직격)

페르미온이 왜 1/2 스핀을 갖는가 — 360도 회전이 항등이 아니고 720도 회전이 항등인 이유. 사원수 회전이 단위 사원수 양쪽 곱 (q v q⁻¹) 구조에서 자동 도출.

공리 1 사원수 명제 (v1.6) 에서 공간 3축 = xi + yj + zk. 사원수 회전 작용은 양쪽 곱 ==> 360도 회전 시 q ==> -q (위상 반전), 720도에서 q ==> q (항등). 공리 1 단위원 형식 명제의 (-1)^n 위상 패턴이 3D 회전에 그대로 확장.

공리 1 (직교 4축) ==> 공리 1 사원수 명제 (3축 ==> 단위구) ==> 공리 1 단위원 형식 명제 (i² = -1 패턴) ==> 공리 4 노름 reading (호 = pi). 720도 = 2 × 360도 = 단위 사원수 두 바퀴 = 위상 (-1)² = +1 항등.

페르미온 = 사원수 표현. 보존 = 벡터 표현. 360도에서 페르미온 위상 -1 (스핀 1/2), 보존 위상 +1 (스핀 1). SU(2) ↔ SO(3) 2:1 이중 덮개 = 같은 구조의 군론 표현.

페르미온 스핀 = 1/2, 3/2, 5/2 (반정수). 보존 스핀 = 0, 1, 2 (정수). 4세대 페르미온 부재 = 사원수 단위 i, j, k 외 4번째 단위 부재 (공리 1 사원수 명제 적용 대상 = 3축뿐).

실험적으로 확인됨: 전자 g ≈ 2 (스핀 1/2), Stern-Gerlach 분리, Werner 1985 중성자 회전 720도 항등 직접 측정.

Pauli 1924 스핀, Dirac 1928 방정식 (사원수 동형), Stern-Gerlach 분리, Werner 1985 중성자 간섭 실험

표준모형은 스핀을 입력 매개변수로 받음. "왜 1/2인가" 답 없음. 반야프레임은 사원수 양쪽 곱 구조의 직접 귀결로 도출.

4세대 페르미온이 발견되면 사원수 3 단위 한계 깨짐 ==> 본 가설 부정. P-03 (4세대 부재) 와 동시 검증.

사원수 회전군 SU(2) ↔ 표준모형 약력 SU(2)_L 의 직접 동형 도출. 현재 H-02 부분 매핑.
